# Plain-Text

> Core repository for plain-text rendering and editing applications.

This repository is a collection of core applications for plain-text rendering and editing. It was developed with a focus on efficient text data processing, Markdown parsing, and implementing an intuitive user interface.

## 📂 Repository Structure

This repository consists of two main projects that operate independently but share the common goal of text processing.

| Project | Description | Status |
| :--- | :--- | :--- |
| **[`LightNote`](./LightNote)** | A lightweight note-taking application. This project focuses on fast text input and management. | `Active` |
| **[`MarkdownEditor`](./MarkdownEditor)** | A Markdown editor supporting real-time rendering. It includes the parsing and visualization architecture for Markdown syntax. | `Active` |

## 📐 Architecture & Design Principles

These projects were built based on the following technical goals and design principles:

* **Modularity:** Improves maintainability by separating the rendering engine from the text input interface.
* **Performance:** Minimizes rendering lag that occurs during large-volume text input.
* **Scalability:** A flexible architectural design that allows for the future addition of various text formats and plugins.
